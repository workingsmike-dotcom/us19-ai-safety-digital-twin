# US 19 Interactive Safety Digital Twin

Interactive corridor-level digital twin for testing roadway safety interventions in real time.

## Features

- Existing vs proposed safety conditions
- AI-based crash risk prediction
- Real-time intervention testing
- 3D risk visualization
- Corridor hotspot identification
- Scenario comparison metrics

## Study Corridor

US 19 corridor between SR 60 and SR 686 in Pinellas County, Florida.

## Technologies

- Python
- Streamlit
- GeoPandas
- PyDeck
- Scikit-learn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
